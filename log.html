Histórico

Em algum momento do desenvolvimento pode ser interessante ver o que já foi modificado nos arquivos. O git pode mostrar o histórico dessas modificações para consultas.

  • Modo terminal
  • Modo gráfico com gitg

Modo terminal

A qualquer momento você pode ver o histórico de mudanças digitando

$ git log

Se você quiser ver as modificações em cada passo, digite

$ git log -p

Frequentemente uma visão geral é útil para sentir o que foi mudado

$ git log --stat --summary

Modo gráfico com gitg

  1. Abra o gitg.

  2. Clique na aba Histórico.

  3. Clique em um item do histórico.

  4. Explore as abas embaixo.
