Colocando seu nome

O git acompanha as mudanças e associa cada mudança a uma pessoa (nome e e-mail). Então, antes de começar a mexer nas coisas, deve-se primeiro colocar o nome.

  • Modo terminal
  • Modo gráfico com gitg

Modo terminal

  1. Digite git config --global user.name "Nome Completo" para mudar o nome.

  2. Digite git config --global user.e-mail user@exemplo.com para mudar o e-mail.

Agora você pode fazer suas modificações e dar os commits.

Modo gráfico com gitg

  1. Vá ao menu Editar ▸ Preferências.

  2. Clique na aba Configuração.

  3. Digite seu nome completo e seu e-mail e clique Fechar.

Agora você pode fazer suas modificações e dar os commits.
