Iniciando um repositório git

Para criar um repositório e o git começar a acompanhar os arquivos do diretório,faça o seguinte:

  1. Vá para esse diretório (com cd diretório).

  2. Digite git init

    $ git init
Initialized empty Git repository in diretório/.git/
