Checando o estado do repositório

A qualquer momento, pode ser útil ver como anda o repositório (quais são os arquivos novos, os modificados etc). Então utilize um dos seguintes procedimentos:

  • Modo terminal
  • Modo gráfico com gitg

Modo terminal

Com o terminal no diretório acompanhado, digite git status.

Modo gráfico com gitg

Com o terminal no diretório acompanhado, digite gitg.
